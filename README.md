# Solo.Life.Journey

A personal blog about tech, self-growth, and everyday life.  
Built with Astro and deployed on Cloudflare Pages.

🌐 [solo-stories.com](https://solo-stories.com)

## Tech Stack

- [Astro](https://astro.build)
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS](https://tailwindcss.com) + [@tailwindcss/typography](https://tailwindcss.com/docs/typography-plugin)
- [rehype-pretty-code](https://rehype-pretty-code.netlify.app)
- [Astro RSS](https://docs.astro.build/en/guides/rss/)
- [Astro Sitemap](https://docs.astro.build/en/guides/integrations-guide/sitemap/)
- [Cloudflare Pages](https://pages.cloudflare.com)
